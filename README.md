# GK PilePro

## Chay ngay

Mo file:

```bat
dist\Tool_KL_UI8.exe
```

Neu can ban Admin:

```bat
dist\GK PilePro Admin.exe
```

## Chay tu source

May can co Python 3.12.

Lan dau:

```bat
install.bat
```

Nhung lan sau:

```bat
run.bat
```

## Cau hinh API

Tao file `.env` tu `.env.example`, roi dien API key/model neu can.

```bat
copy .env.example .env
```

File `.env` khong duoc dua len Git.
